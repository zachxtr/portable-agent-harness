# Sample reference implementation

> **Reference only** — not installed by `./scripts/init.sh`.  
> Compare this mature harness to the sanitized files in [`../../template/`](../../template/).

This example is a real-world harness from an active product codebase (agent folder **`joshua/`**, rich `memory/`, historical `code-log/`). Use it to see:

- How `MEMORY.md` looks after many sessions
- WIP plans, architecture depth, and project-specific skills
- Code log tone and structure at scale

When bootstrapping a new project, use the **template** via `init.sh`, not this folder.
