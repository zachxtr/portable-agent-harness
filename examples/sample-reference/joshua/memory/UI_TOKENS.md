# UI Color Terminology (locked)

Dashboard and marketing color tokens — reference when touching UI theming.

| Token | Meaning |
|-------|---------|
| `highlight-*` | Fixed brand teal — marketing, auth, legal pages |
| `accent-*` | Per-user dashboard interactive theme (500 = primary fill, 600 = link/darker) |
| `annot-*` | Fixed PDF annotation tool colors |
| `pill-tag-*` / `pill-filter-*` | Shared gray-300/gray-700 chip resting state; filters hover gray-200 |

**Accent roles (dashboard):** tint (8% primary bg), primary (buttons/fills), link (text/icons), selected (filter-on bg), soft (borders/rings). No separate muted/deep steps.
